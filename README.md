# 🌍 Roamora Travel

A modern and responsive travel booking website built using HTML, CSS, and JavaScript.

## ✨ Features

### 👤 User Features
- Browse travel destinations
- Search travel packages
- Book trips
- Submit inquiries
- Contact support
- Responsive design

### 🛠️ Admin Features
- Admin login
- Manage bookings
- View customer details
- Track revenue
- Manage inquiries
- Edit and delete records

## 💻 Technologies Used

- HTML5
- CSS3
- JavaScript
